MovieCritics is a desktop-based movie management system designed specifically for family environments. It allows users to manage their movie libraries, track watch history, and provides a safe browsing experience for younger family members through role-based access.

🚀 Features
Role-Based Access Control: Distinct interfaces and permissions for Parent and Child roles.

Parental Controls: Automatic filtering of sensitive content to ensure child safety.

Movie Management: Full CRUD (Create, Read, Update, Delete) operations for movie records (Parent only).

Interactive Review System: Rate movies on a scale of 1–10 and leave detailed comments.

Search & Filter: Find movies by title, genre, year, or director.

Personal Watchlist: Track watched movies and manage personal progress.

Analytics Dashboard: Visual insights into family movie-watching habits.

🛠️ Tech Stack
Language: Java (JDK 22)

Database: MySQL 8.0

IDE: Apache NetBeans 20

Modeling: Visual Paradigm (UML diagrams)

Standards: Prepared based on IEEE Std 830-1998 (SRS)

📋 Requirements
To run this project locally, you will need:

Java Runtime Environment (JRE) / JDK 22 or higher.

A running MySQL server instance.

Necessary JDBC drivers for database connectivity.

📂 Project Structure
src/: Java source files for GUI and Logic.

database/: SQL scripts for setting up the database schema.

documentation/: Full SRS report and UML diagrams.
